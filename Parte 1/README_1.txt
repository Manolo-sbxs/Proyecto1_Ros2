# Parte 1 – Modelo URDF

En esta carpeta se encuentra el archivo "triciclo.urdf", el cual corresponde al primer paso en la confección del robot tipo triciclo. Este modelo define la estructura base del robot en formato URDF, incluyendo chasis, ruedas y horquilla. 

El robot cuenta con tres grados de libertad, siendo estos la propulsión de ambas ruedas traseras como el primero, la horquilla como el segundo y la rueda delantera libre como el tercero.

Este mismo archivo será utilizado en las siguientes dos etapas del proyecto, la modularización de Xacro y visualización en RViz.
